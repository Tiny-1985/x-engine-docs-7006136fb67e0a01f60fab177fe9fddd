#### 使用方式

\- `yarn install 安装依赖` 

\- `yarn dev 运行示例`    

\- `yarn build 打包模块` 





\---



####  安装xengine.js引擎



\- yarn安装

```
yarn add https://github.com/littlegrass003/zkty_engine.git
```



\- npm安装

```
npm install https://github.com/littlegrass003/zkty_engine.git
```
