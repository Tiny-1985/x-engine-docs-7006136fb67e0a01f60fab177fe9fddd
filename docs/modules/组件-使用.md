# iOS 平台可选模块接入指引

以蓝牙组件为例.
从 组件仓库 里找到相应的组件.


module.json

```
{
  "module_id":"com.zkty.bluetooth",
  "name":"蓝牙",
  "name_space": "_ui",
  "tag": "1.0.0",
  "engine_version": "1.0.0",
  "minimal_os_version":{
		"ios": "11.1.0",
  	"android_api": 20,
	}
}
```

将上述文件复制到 xengine_config.json 的 modules 下:

```json
{
  ...
  "modules":[
    {
      "module_id":"com.zkty.bluetooth",
      "name":"蓝牙",
      "tag": "1.0.0",
      "engine_version": "1.0.0",
      "minimal_os_version":{
        "ios": "11.1.0",
        "android_api": 20
    }  
  }
}
```
