
## 集成流程

以蓝牙组件为例.
从 组件仓库 里找到相应的组件.


module.json

```
{
  "module_id":"com.zkty.bluetooth",
  "name":"蓝牙",
  "tag": "1.0.0",
  "engine_version": "0.0.1",
  "mini_platform":{
		"ios": "11.1.0",
  	"android_api": 20,
	},
	"exclude_brand":[
    {
      "brand_name":"华为",
      "device_id":"ry-10"
    }
  ]
}
```

将上述文件复制到 xengine_config.json 的 mdules 下:

```json
{
  "modules":[
    {
      "module_id":"com.zkty.bluetooth",
      "name":"蓝牙",
      "tag": "1.0.0",
      "engine_version": "0.0.1",
      "mini_platform":{
        "ios": "11.1.0",
        "android_api": 20
      },
      "exclude_brand":[
        {
          "brand_name":"华为",
          "device_id":"ry-10"
        }
      ]
    }
  ]
}
```
