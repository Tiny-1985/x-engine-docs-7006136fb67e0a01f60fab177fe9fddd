# 组件 - navigator原生导航

- 我们为您提供了一系列的原生(iOS, Android)导航的跳转以及设置导航条左边和右边的按钮.
- 一定程度上提高用户体验, 如果页面无响应, 用户可以返回. 不至于只能杀死进程.

## API 例表

### setNavTitle

- 设置导航栏中间名称

- 函数声明
  
  - navigator.setNavTitle(Object)
  

**Object参数说明**	

|    参数    |  类型  | 必填 | 默认值  |       说明       |
| :--------: | :----: | :--: | :-----: | :--------------: |
|   title    | String |  是  |   ---   |   导航条的文字   |
| titleColor | String |  否  | #000000 | 16进制的颜色色值 |
| titleSize  | Number |  否  |   16    | 导航条文字的大小 |

**示例**	

```javascript
xengine.navigator.setNavTitle({
  title:'导航条',
  titleColor : "#000000",
  titleSize  : 16
});
```

### setNavRightBtn

- 显示操作菜单
- 函数声明
  
  - navigator.setNavRightBtn(Object)
  
  

**Object参数说明**	

|   参数   |     类型      | 必填 | 默认值 |           说明           |
| :------: | :-----------: | :--: | :----: | :----------------------: |
| itemList | Array<string> |  是  |  ---   | 按钮的文字数组,最大长度6 |
|  title   |    string     |  否  |  ---   |        提示的标题        |
| content  |    string     |  否  |  ---   |        提示的内容        |
| success  |   Function    |  否  |  ---   |  接口调用成功的回调函数  |
|   fail   |   Function    |  否  |  ---   |  接口调用失败的回调函数  |

**success返回参数说明**

| 参数     | 类型   | 说明                                |
| -------- | ------ | ----------------------------------- |
| tapIndex | Number | 用户点击的按钮,从上到下顺序,从0开始 |

**示例**	

```javascript
// success / fail
xengine.ui.showActionSheet({
  itemList: ['A', 'B', 'C', 'D', 'E'],
  success: function (res) {
    console.log(res)
  },
  fail: function (error) {
    console.log(error)
  }
})

--------------

// promise
xengine.ui.showActionSheet({
  itemList: ['A', 'B', 'C', 'D', 'E'],
}).then((res) => {
  console.log(res)
}).catch((error) => {
  console.log(error)
})
```

### navigatorPush

- 显示模态弹窗，类似于标准 html 的消息框：alert、confirm。
- 函数声明
  - navigarot.navigatorPush(Object)

**Object参数说明**	

|    参数    |   类型   | 必填 | 默认值 |          说明          |
| :--------: | :------: | :--: | :----: | :--------------------: |
|   title    |  String  |  是  |  ---   |       提示的标题       |
|  content   |  String  |  否  |  ---   |       提示的内容       |
| showCancel | Boolean  |  否  |  true  |    是否显示取消按钮    |
|  success   | Function |  否  |  ---   | 接口调用成功的回调函数 |
|    fail    | Function |  否  |  ---   | 接口调用失败的回调函数 |



**success返回参数说明**

| 参数     | 类型   | 说明                 |
| -------- | ------ | -------------------- |
| tapIndex | Number | 取消返回0, 确定返回1 |

**示例**	

```javascript
// success / fail
xengine.ui.showModal({ 
  title: 'title',
  success: function (res) {
    console.log(res)
  },
  fail: function (error) {
    console.log(error)
  }  
})

--------------

// promise
xengine.ui.showModal({
  title: 'title' 
}).then((res) => {
  console.log(res)
}).catch((error) => {
  console.log(error)
})
```

### navigatorBack

- 显示消息提示框。
- 函数声明
  - navigator.navigatorBack(Object)

**Object参数说明**		

|   参数   |   类型   | 必填 | 默认值 |            说明             |
| :------: | :------: | :--: | :----: | :-------------------------: |
|  title   |  String  |  是  |  ----  |         提示的标题          |
| duration |  number  |  否  |  2000  | 提示的延迟时间.默认：2000ms |
|   icon   |  String  |  否  |  ---   |    图标,有效值见下方说明    |
| success  | Function |  否  |  ---   |      接口成功回调函数       |
|   fail   | Function |  否  |  ---   |      接口失败回调函数       |

**icon值说明**

| 参数    | 说明         |
| ------- | ------------ |
| success | 显示成功图标 |
| loading | 显示加载图标 |
| none    | 不显示图标   |

**示例**	

```javascript
// success / fail
xengine.ui.showToast({
  title: 'title',
  icon: 'success',
  duration: '2000',
  success: (res) => {
    alert(res);
  },
  fail: (error) => {
    alert(error); 
  }
});


// Promise
const params = {
  title: 'title',
  icon: 'success',
  duration: '2000', 
}
xengine.ui.showToast(params).then((res)=>{
  alert(res)
}).catch((error) => {
  alert(error)  
})
```
