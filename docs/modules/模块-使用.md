# 模块接入指引

以蓝牙模块为例.
从 模块仓库 里找到相应的模块.

## 手动方式

在 pod 里引用。

Podfile:

```config
 pod "com.zkty.bluetooth:1.0.0"
```

build.gradle:

```build.gradle
dependencies {
		...
    compile 'com.zkty.bluetooth:1.0.0'
		...
}

```

package.json:

```json
dev:{
  "com.zkty.bluetooth":"1.0.0"
}
```

从私库里拉取各自的包.
