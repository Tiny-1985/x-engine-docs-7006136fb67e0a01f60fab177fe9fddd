微应用完全尊重 H5。所以你可以用任何你喜欢的前端框架开发。

但推荐使用 Taro， 这样你就也可以发小程序端了。



包格式如下：

- {microAppId}.{version}.zip
  - index.html
  - icon.png 
  - xapp.json



## 语言与框架要求

微应用对前端框架没有要求。 但我们会提供一些框架的注意事项。



###  vue 注意事项

推荐使用开源的 UI 库， 如 mand，vant， 等。

1. 历史页面的 tab 在回退时能够保持状态。 可以使用 keep-alive
